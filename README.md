## 🛠 Stack technique

| Couche       | Technologies                                     |
| ------------ | ------------------------------------------------ |
| Framework    | React 18 + TypeScript                            |
| Build        | Vite 5                                           |
| Styling      | Tailwind CSS 3 + shadcn/ui                       |
| Animations   | Framer Motion                                    |
| Routing      | React Router 6                                   |
| Tests        | Vitest + Testing Library                         |

> Projet 100 % frontend — pas de backend ni de base de données.

## 📂 Structure du projet

```
src/
├── assets/            # Images (hero background, etc.)
├── components/        # Composants React
│   ├── ui/            # Composants shadcn/ui
│   ├── Hero.tsx       # Section d'accueil
│   ├── Navbar.tsx     # Navigation
│   ├── FeaturedProjects.tsx  # Projets phares
│   ├── AllProjects.tsx       # Tous les projets (filtrable)
│   ├── Skills.tsx     # Compétences techniques
│   ├── Experience.tsx # Parcours professionnel
│   └── Contact.tsx    # Formulaire de contact
├── data/
│   └── projects.ts    # Données statiques (projets, compétences, expériences)
├── pages/
│   ├── Index.tsx      # Page principale (single-page)
│   └── NotFound.tsx   # Page 404
└── hooks/             # Custom hooks
public/
├── CV_Jaad_CHOKR.pdf # CV téléchargeable
└── favicon.ico
```
